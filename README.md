 🚌 EFOY Bus Booking System

The **EFOY Bus Booking System** is a web-based application designed to make bus ticket booking simple, fast, and user-friendly. It allows users to search for available buses, view routes and schedules, and book tickets online.

---

 🚀 Features

 🧭 **User-Friendly Interface** — Simple and intuitive design for easy navigation.
 🚌 **Bus Schedule Management** — View available buses, departure times, and destinations.
 💳 **Online Ticket Booking** — Reserve bus seats quickly and securely.
 🧾 **Booking History** — Users can view their previous bookings.
 🔄 **Refund System** — Easy refund request and tracking.

---

 🧠 Technologies Used

 **Frontend:** HTML, CSS, JavaScript
 **Backend:** PHP (CodeIgniter 4 Framework)
 **Database:** PostgreSQL

---

 ⚙️ Setup Instructions

1. **Clone the Repository**

   ```bash
   git clone https://github.com/Ahla22-19/EFOY-BUS-BOOKING-SYSTEM.git
   ```

2. **Navigate into the Project Folder**

   ```bash
   cd EFOY-BUS-BOOKING-SYSTEM
   ```

3. **Set Up Your Local Server**

   * Place the project inside your XAMPP `htdocs` folder (or similar for other servers).
   * Start Apache and MySQL.

4. **Import the Database**

   * Open **phpMyAdmin**.
   * Create a new database (e.g., `efoy_bus`).
   * Import the provided `.sql` file (if available).

5. **Run the Application**

   * Open your browser and go to:

     ```
     http://localhost/EFOY-BUS-BOOKING-SYSTEM
     ```
